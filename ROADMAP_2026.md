- Navigation (header, footer, menu mobile)
- Sections (hero, features, cta, gallery)

**Outils :**
- Figma pour les maquettes
- Storybook pour les composants (optionnel)

### 📋 Août : Process Client
Documenter votre workflow :
1. **Brief client** (questionnaire)
2. **Devis automatisé** (template)
3. **Maquette Figma** (validation)
4. **Développement** (checklist)
5. **Tests** (mobile, desktop, formulaires)
6. **Formation client** (utiliser le CMS)
7. **Livraison** (accès CMS + domaine)
